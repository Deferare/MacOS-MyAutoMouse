# MyAutoMouse

macOS에서 자동 클릭/마우스 이동 기능을 구현하기 위한 Xcode 프로젝트입니다.

## 개발 환경

- Xcode (macOS 최신 버전 권장)
- Swift / SwiftUI

## 시작하기

1. `MyAutoMouse/MyAutoMouse.xcodeproj`를 Xcode에서 엽니다.
2. 실행 타겟을 macOS로 선택합니다.
3. `Run`으로 앱을 실행합니다.

## Git 기본 규칙

- 사용자별/로컬 파일(`xcuserdata`, `*.xcuserstate`, `.DS_Store`)은 커밋하지 않습니다.
- 빌드 산출물(`DerivedData`, `build`)은 커밋하지 않습니다.
